# Collect Service

## Запуск проекта

1. Клонировать репозиторий:
git clone https://github.com/knowlegediplom-star/django_project

2. Перейти в папку:
cd collect_service

3. Запустить:
docker-compose up --build

4. Открыть:
http://127.0.0.1:8000/api/docs/

## Описание

Реализован сервис денежных сборов с API, Swagger, тестами и Docker.
